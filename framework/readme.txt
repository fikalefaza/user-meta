
===== Convension =====

== General ==
* To avoid confusion, all variable should be singular number. eg: $field not $fields /// Not applicable


== PHP ==
* wp recognized varable should be undercase. e.g: $user_field
* custom varaible or method may be capitalized e.g: $userField


== JS ==
* Function name should be Capitalized and start with prefix. eg: umNewField()
* Variable name should be capitalized, eg: newID


== CSS ==
* ID and CLASS should be underscored and start with prefix. eg: um_admin_sidebar
